 # Data Structure

This repository contains **C++ implementations** of fundamental data structures with **interactive console menus**:

- **Queue** – Enqueue, Dequeue, Display, Head/Tail values, Update value  
- **Stack** – Push, Pop, Display, Top value, Update value  
- **Linked List** – Insert, Delete, Search, Display (can be added later)

Each program is designed to follow a **menu-driven approach**, where the user can select operations and perform them step by step.

## Features

- Fully **interactive console menu**  
- Proper **display formatting** for better visualization  
- Update functions to modify values in Stack or Queue
